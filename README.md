# Calc4Dent - Polish Dentists' Calculator
Calc4Dent is a simple calculator application designed for Polish dentists to help them with various calculations related to the National Health Fund (NFZ) points system. It allows dentists to input procedure codes and calculate their corresponding NFZ points. Additionally, the app keeps a record of all performed actions and provides a list of all available procedure codes.

## Features
- Input procedure codes and calculate NFZ points.
- Keep a history of performed actions for future reference.
- Access a comprehensive list of all available procedure codes.
